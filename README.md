<h1>예약 서비스 수요가 늘어남에 따라 단체예약을 한 번에 도와주는 서비스 단체</h1>
 <p>예약 시 전화로 예약하는 불편함으로 단체 예약이 가능한 업체와 바로 연결하여</p>
 <p>실시간 예약이 가능하게 하는 서비스, 단체 예약 시 할인, 제약 등 정보제공</p>
</div>
<br>

 주요 기능 
- 단체 예약이 가능한 사업장과 연결하여 실시간, 날짜별 예약이 가능한 서비스를
 제공
 > 예약페이지에서 예약 정보 입력 후 예약/결제 예약확인 구현
 > 예약가능한 장소의 위치 제공
- 챗봇을 이용한 고객상담과 간편예약, 정보제공 서비스
 > 단체예약에 필요한 정보와 예약은 기본 질문 선택할 수 있게 배치 
 > 담당 관리자와 상담을 원하는 경우 실시간 채팅상담 제공 

 담당업무
- 업체 리스트 페이지 구현
- 업체 상세 정보 페이지 구현
- 사업자 회원가입 및 로그인 기능
- 업체 검색 기능
- 토스페이 API 구축 및 활용
- Oracle 테이블 구축
- Oven 자료 제작
